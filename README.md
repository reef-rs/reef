# cargo-reef

CLI scaffolder for [Reef](https://reef.rs) apps.

```bash
cargo install cargo-reef
cargo reef new my-app
```

> ⚠️ **Early development.** This repository is being bootstrapped from PRs.
> Initial scaffolding lives on the `setup` branch and will be merged to `main`
> after review.
